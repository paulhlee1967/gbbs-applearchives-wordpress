A retro Apple II-inspired WordPress block theme that authentically recreates the classic Apple II computer interface with modern WordPress functionality. Features monochrome terminal styling, authentic Apple II fonts, ASCII art elements, and full site editing support.

== Description ==

The GBBS Pro theme brings the nostalgic charm of the classic Apple II computer to your WordPress site. With its green-on-black color scheme, monospace typography, and terminal-style UI elements, it's perfect for retro computing enthusiasts, developers, and anyone who appreciates vintage aesthetics.

**Key Features:**
* Authentic Apple II design with monochrome terminal styling
* Custom Apple II font (PrintChar21)
* ASCII art separators and terminal-style components
* Full Site Editing (FSE) compatible block theme
* Complete template set (index, page, archive, search, 404)
* Modular template parts (header, footer, loop, comments)
* Responsive design with mobile-first approach
* Accessibility ready with WCAG compliance
* Performance optimized with modern web standards

**Perfect for:**
* Retro computing blogs and websites
* Developer portfolios and technical documentation
* Nostalgic personal websites
* Terminal and command-line enthusiasts
* Vintage computing communities

== Installation ==

1. Upload the theme files to `/wp-content/themes/gbbs-pro/`
2. Activate the theme in WordPress Admin → Appearance → Themes
3. Customize using the Site Editor (Appearance → Editor)

== Frequently Asked Questions ==

= Does this theme work with the block editor? =
Yes! This is a full block theme that takes full advantage of WordPress's Full Site Editing capabilities.

= Are the Apple II fonts included? =
Yes, the theme includes the authentic PrintChar21 font in both WOFF2 and TTF formats.

= Is the theme responsive? =
Absolutely! The theme uses a mobile-first responsive design approach while maintaining the Apple II aesthetic.

= Can I customize the colors? =
Yes, you can customize the color palette through the Site Editor or by editing the theme.json file.

= Does it support accessibility? =
Yes, the theme includes skip links, proper ARIA labels, focus indicators, and screen reader support.
